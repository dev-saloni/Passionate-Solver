# Passionate Solver 🔥

A collection of coding problems and solutions I've been working on throughout my journey as a CS student. This repo has everything from basic logic problems to some DSA stuff I've practiced over time.

---

## About

Honestly, I started this repo just to keep track of all the problems I was solving and not lose my work. It turned into something I actually enjoy maintaining. The problems here are mostly in **Java**, **Python**, and some small **web dev** stuff using HTML, CSS, and JavaScript.

If you're also learning and want to see how I approached a problem — feel free to look around!

---

## What's in here

- Java programs (OOP, logic, practice problems)
- Python scripts and exercises
- Some SQL queries I've worked on
- Basic web dev mini exercises (HTML/CSS/JS)
- Random problem-solving stuff I found interesting

## How to run

```bash
git clone https://github.com/dev-saloni/Passionate-Solver.git
cd Passionate-Solver
```

For Java files:
```bash
javac FileName.java
java FileName
```

For Python files:
```bash
python filename.py
```
